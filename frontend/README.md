# TeamClaw Frontend Prototype

## 说明

这是一个可点击跳转的前端原型，使用 Vue3 + Vue Router + Vuetify CDN 方式实现。

## 运行方式

1. 在仓库根目录执行：
   - `python3 -m http.server 8080`
2. 浏览器打开：
   - `http://localhost:8080/frontend/`

## 已实现页面

- 登录/注册/忘记密码
- 新建聊天
- 聊天对话（消息流、审批卡片、任务面板）
- Skills 列表
- Skill 编辑器
- 个人信息
- 管理控制台（含工具审计日志）
